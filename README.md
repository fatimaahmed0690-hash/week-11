# Week 11 Cybersecurity Platform (OOP Refactor)

## Features
- Login/Register (User class)
- Main Dashboard (SecurityIncident class)
- Threat Intelligence module
- AI Security Assistant (OpenAI GPT integration)

## Structure
- "classes.py": Core OOP classes (User, SecurityIncident, Dataset, ITTicket)
- "database_manager.py": MySQL database handler
- "pages/": Streamlit pages
- ".env": Environment variables (OpenAI API key, DB credentials)

## Setup
1. Install dependencies:
   pip install -r requirements.txt
